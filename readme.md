# Merge PDF files into one

This is a small python GUI program that allows to merge multiple PDF files into one.

## Requirements

### Make sure that you have a python with `tkinter` installed.

```sh
python -m tkinter
```
Should display a tiny window with a QUIT button.
If not - install python with tkinter.

### Install required libs.

```sh
pip install -r requirements.txt
```

## Usage

Run a `main.py` script. Select files. Merge selected. Export merged file to a preferred destination.

```sh
python main.py
```
